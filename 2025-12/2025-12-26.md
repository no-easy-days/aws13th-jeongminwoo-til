## 변수와 자료형 
-자료형 : 자료(데이터)를 기억장소에 저장하고 활용할 때 표현하는 타입(형)
| 종류 | 설명 | 예시 |
|------|------|------|
| int | • 정수 자료형이며 양수,음수의 부호를 가짐 | 110,-20,0 |
| float | • 실수 자료형으로 부동소수점 수<br>• 크기는 8byte| 12.5,-0.44 |
| bool | • 참,거짓 두가지 값을 표현하는 방식의 값을 처리 | True,False |
| str | • 문자열 자료형으로 큰따옴표나 작은따옴표로 둘러줌 | 'korea', "korea" , "1234" |
| list | • 리스트 자료형으로 순서를 가짐 | [1.2,'string',False]|
| set | • 순서가 없는 집합형 자료형이며 중복을 허용하지 않음<br> • set은 수정이 가능 | {1.3,'string'} |
| tuple | • 혼합 형태의 자료를 기억할 수 있는 자료형으로 읽기 전용 | (1.0,'string',True) |
| dict | • 연관 배열로 키와 값이 쌍으로 구성된 집합형 자료형 | {'key1':'value1','key2':'value2'} |

## 연산자 
### 산술 연산자 
| 연산자 | 기능 | 사용 예 | 
|------|------|------|  
| `+` | 더하기 | `c += a` | 
| `-` | 빼기 | `c -= a` |
| `*` | 곱하기 | `c *= a` |
| `/` | 나누기 | `c /= a` |
| `**` | 거듭제곱 | `c **= a` |

### 관계 연산자 
- 반환값은 True 또는 False

| 관계 연산자 | 기능 |
|------------|------|
| `A > B` | A가 B보다 크다 |
| `A < B` | A가 B보다 작다 |
| `A >= B` | A가 B보다 크거나 같다 |
| `A <= B` | A가 B보다 작거나 같다 |
| `A == B` | A와 B가 같다 |
| `A != B` | A와 B가 같지 않다 |

### 비트 연산자 

| 비트 연산자 | 기능 |
|------------|------|
| `A & B` | AND 연산 (둘 다 1이면 1) |
| `A \| B` | OR 연산 (하나라도 1이면 1) |
| `A ^ B` | XOR 연산 (서로 다르면 1) |
| `~A` | NOT 연산 (비트 반전) |
| `A << B` | 왼쪽 시프트 (A를 B비트만큼 왼쪽 이동) |
| `A >> B` | 오른쪽 시프트 (A를 B비트만큼 오른쪽 이동) |
