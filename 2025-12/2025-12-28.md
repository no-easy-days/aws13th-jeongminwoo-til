## 조건문 if
1. 기본if 문  
```python
 if 조건식:
   수행문 1
   ...
   수행문 n 
```
2. if~else 문
```python
if 조건식:
  수행문 1
  ...
else:
  수행문 2
  ...
```
3. 다중if 문
```python
if 조건식1:
  수행문 1
elif 조건식2:
  수행문 2
elif 조건식3:
  수행문 3
else:
  수행문
```
## 반복문 while 
- 조건을 만족하는 동안 수행문을 반복해서 처리하고 while 조건식이 거짓이 되면 반복처리가 종료  
### 기본 while문
```python
a = 1
while a <=5:
   print(a,end=' ')
   a += 1
print(a)
```  
### 중첩 while문
- 반복문 while 블록 안에 또 다른 while문을 사용하며, 안쪽 while문을 벗어나고 바깥쪽 while문까지 벗어나면 전체 반복 작업이 끝남
```python
i=1
while i <= 2:
 j = 1
 while j <= 3:
   print('i:' + str(i) + 'j=' + str(j))
   j += 1
i += 1
print()
```
### continue, break을 포함한 while문 
- continue : 반복문을 끝내지 않음 , 현재 반복만 스킵하고 다음 반복으로 이동
- break : while문을 즉시 종료, 조건이 남아있어도 강제로 탈출
```python
a = 0

while a<10:
   a += 1
   if a == 5: continue
   if a == 7: break
   print(a)
else:
  print('종료')

print('while 수행 후 %d'%a)
```

### 여러 형식의 for문 
- for문으로 리스트, 딕셔너리 자료형 처리
```python
price = {'사과': 2000, '감': 500, '바나나': 1000}
my = {'사과':2,'감':3}
imsi = (f for f in my)
for i in imsi:
   print('키:',i)
   print('값:price는{},my{}는'.format(price[i],my[i]))

bill = sum(price[f]*my[f] for f in my)
print('구매한 과일값은 {}원'.format(bill))
```
