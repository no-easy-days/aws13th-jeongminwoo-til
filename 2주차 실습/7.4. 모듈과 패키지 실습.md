# 7.4. 모듈과 패키지 실습 

## 문제 1: import 기본 사용 
```python
import datetime

today = datetime.date.today()
print(today)
```
## 문제 2: from import 사용 
```python
from math import sqrt,pow

print(int(sqrt(16)))
print(int(pow(2,10)))
```
## 문제 3: 별명(as) 사용 
```python
import random as rd

print(rd.randint(1, 100))
```
